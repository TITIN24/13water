##蔡峰和陈金杰的十三水软件工程结对作业
###运行环境
兼容各种浏览器，不需要多余的运行环境，打开网页就能嗨。

***

###编译方法
不需要编译，游戏可以直接运行，不需要安装多余的东西。

***

###使用方法
首先打开浏览器  
新用户注册账号，老用户直接登录  
进入游戏就可以进入牌局  
对战界面右上角可以打开背景音乐  
同样设有排行榜、历史战绩等，均在右上角  
历史战绩详情由历史战绩中点击查看  
如果想要退出游戏，在对战界面点击图标退出哦
